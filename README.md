## Big Data Masters Project
### Dataset
US Accidents Dataset: https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents

#### Description
- **Title**: US Accidents Dataset
- **Coverage**: 49 states of the USA
- **Time Period**: February 2016 - March 2023
- **Number of Records**: About 7.7 million accident records
